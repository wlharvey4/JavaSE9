B.3.1.7 Javadoc Doclet Options

The javadoc tool supports documentation comments in module declarations. It has command-line options, such as

  • --module-path,
  • --upgrade-module-path, or
  • --module-source-path

to configure the set of modules to be documented, and generates a new summary page for any modules being documented. The module-related options are available for generating documentation.
