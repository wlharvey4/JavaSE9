A.2 The General Form of a Documentation Comment

After the beginning /**, the first line or lines become the main description of your class, interface, field, constructor, method, or module. After that, you can include one or more the various @ tags. Each @ tag must start at the beginning of a new line or follow one or more asterisks (**) that are at the start of a line. Multiple tags of the same type should be grouped together. In-line tags (thse that being with a brace ({) can be used within any description.
