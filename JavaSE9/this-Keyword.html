2.5 The this Keyword

Sometimes a method will need to refer to the object that invoked it. To allow this, Java defines the this keyword. this can be used inside any method to refer to the current object. That is, this is always a refernce to the object on which the method was invoked. You can use this anywhere a reference to an object of the current class’ type is permitted.
