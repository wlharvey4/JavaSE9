Appendix A Using Java’s Documentation Comments

Java supports three types of comments.

  • //
  • /* */
  • /** */

The third is called a documwentation comment. Documentation comments allow you to embed information about your program into the program itself. You can then use the javadoc utility progrm to extract the information and put it into an HTML file. Documentation comments make it convenient to document your programs. Beginning with JDK 9,m javadoc includes support for modules.
