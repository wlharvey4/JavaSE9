F.7.3 Makefile WorldClean

<Makefile WORLDCLEAN> ≡ 

.PHONY: worldclean
worldclean :
    for file in ${ROOT}/*; do \
      [ \
        $${file} != "${ROOT}/${FILENAME}.twjr" -a \
        $$file != "${ROOT}/Makefile" -a \
        $$file != ".git" \
      ] \
      && rm -rfv "$${file}" || :; \
    done

This chunk is called by <Makefile CLEAN Targets>; see its first definition at Makefile-CLEAN-Targets.
