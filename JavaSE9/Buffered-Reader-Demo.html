15.8.8 Buffered Reader Demo

This example reworks the BufferedInputStream example (see section Buffered Input Example) so that it uses a BufferedReader character stream rather than a buffered byte stream. It uses the mark() and reset() methods to parse a stream for the HTML-entity reference for the copyright symbol. Such a reference begins with an ampersand (&) and ends with a semicolon (;) without any intervening whitespace. The sample has two ampersands to show the case where the reset() happens and where it does not.
