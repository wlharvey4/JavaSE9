7.6.2.2 CopyFileMultTryWR Manage Two Files

Note how the input and output files are opened within the try block. After this block ends, both fin and fout will have been closed. This code is much shorter. The ability to streamline source code is a side-benefit of automatic resouce management.

<CopyFileMultTryWR Manage Two Files> ≡ 

// Open and manage two files via the try statement.
try (FileInputStream fin = new FileInputStream(args[0]);
     FileOutputStream fout = new FileOutputStream(args[1])) {

  do {
    i = fin.read();
    if (i != -1) fout.write(i);
  } while (i != -1);

} catch (IOException e) {
  System.out.println("I/O Error: " + e);
}
