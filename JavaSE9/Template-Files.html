B.2.4.2 Template Files

If you want a template file to be in the source directory, but not generate errors when you execute the javadoc tool, then give it an invalid file name such as Buffer-Template.java to prevent it from being processed. The javadoc tool processes only source files with names, when stripped of the .java suffix, that are valid class names.
