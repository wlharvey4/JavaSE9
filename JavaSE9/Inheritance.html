4 Inheritance

Inheritance is a cornerstone of object-oriented programming because it allows the creation of hierarchical classifications. Using inheritance, you can create a general class taht defines traits common to a set of related items. This class can then be inherited by other, more specific classes, each adding those things that are unique to them.

A class that is inherited is called a superclass. The class that does the inheriting is called a subclass. A subclass is a specialized version of a subclass. It inherits all of the members defined by the superclass and adds its own, unique elements.
