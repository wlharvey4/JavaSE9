25.2.2.4 TYPE Field for Primitive Type Wrappers

The .class syntax is a more convenient and the preferred way to obtain the Class for a primitive type; however, there is another way to acquire the Class. Each of the primitive types and void has a wrapper class in java.lang that is used for boxing of primitive types to reference types. Each wrapper class contains a field name TYPE which is equal to the Class for the primitive type being wrapped.

Wrapper TYPE for Primitive

Class c = Double.TYPE;

There is a class java.lang.Double which is used to wrap the primitive type double whenever an Object is required. The value of Double.TYPE is identical to that of double.class.

Wrapper TYPE for void

Class c = Void.TYPE;

Void.TYPE is identical to void.class.
