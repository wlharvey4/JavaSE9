A.1.21 {@value}

{@value} has two forms. The first displays the value of the constant that it precedes, which must be a static field. It has this form: 

{@value}

The second form displays the value of a specified static field. It has the form: 

{@value ‘pkg.class#field’}

‘pkg.class#field’ specifies the name of the static field.
