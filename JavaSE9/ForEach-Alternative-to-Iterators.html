13.4.2 The For-Each Alternative to Iterators

If you won’t be modifying the contents of a collection or obtaining elements in reverse order, then the for-each version of the for loop is often a more convenient alternative to cycling through a collection than is using an iterator. The for can cycle through any collection of objects that implement the Iterable interface. Because all of the collection classes implement this inteface, they can all be operated upon by for.
