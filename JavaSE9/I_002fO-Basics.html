7.1 I/O Basics

Most real applications of Java are not text-based, console programs. Rather, they are either graphically oriented programs that rely on one of Java’s graphical user interface (GUI) frameworks, such as Swing, the AWT, or JavaFX, for user interaction, or they are Web applications. Text-based console programs do not constitute an important use for Java in the real world. Java’s support for console I/O is limited and somewhat awkword to use. Text-based console I/O is just not that useful in real-world Java programming.

Java does, however, provide strong, flexible support for I/O as it relates to files and networks. Java’s I/O system is cohesive and consistent. A general overview of I/O is presented here. A detailed description is found in chapters describing the Java Library: See section Input/Output — Exploring java.io and See section Exploring NIO.
