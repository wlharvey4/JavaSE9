15.2.1 File Methods

File defines many methods that obtain the standard properties of a File object.

getName()

returns the name of the file

getParent()

returns the name of the parent directory

getPath()
getAbsolutePath()

returns the path

exists()

returns true if the file exists, false if it does not

canWrite()
canRead()

returns whether the file is writeable/readable

isDirectory()

returns whether the file is a directory

isFile()

returns whether the file is a regular file (true) or a non-file (false) such as directory, device drivers, named pipes, etc.

isAbsolute()

returns whether the file is an absolute path (true) or a relative path (false)

lastModified()

returns the modification date and time

length()

returns the file’s size
