A.4 An Example that Uses Documentatation Comments

Following is a sample program that uses documentation comments. Notice the way each comment immediately precedes the item that it describes. After begin processed by javadoc, the documentation about the SquareNum class will be found in SquareNum.html.

SquareNum.java

{SquareNum.java} ≡ 

<Import java.io>

<SquareNum—SquareNum Class>
